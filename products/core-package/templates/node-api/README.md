# Node/API Template (Starter Kit v0.3)

Базовый шаблон для Node.js API на Express.
Цель — дать чистый, понятный фундамент для реальных проектов.

## Быстрый старт

1. Скопируй `.env.example` → `.env`
2. Установи зависимости:
   - npm i
3. Запусти:
   - npm run dev

## Проверка

GET http://localhost:4000/api/health
